# CSV Order Tracking Starter

Ultra barebones starter repo for a 30 minute vibe coding workshop.

## Run

```bash
npm install
npm run dev
```

Then open `/orders`.

## Data

- CSV lives in `data/orders.csv`
- App reads and appends rows directly to that file

## API

- `GET /api/orders`
- `POST /api/orders`

## Workshop flow

Pick a feature from `FEATURES.md` and build from there.
